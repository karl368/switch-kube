# 极简 Kubernetes 集群切换工具 - switch-kube.ps1

## 项目简介
`switch-kube.ps1` 是一个简洁高效的 PowerShell 脚本，旨在简化 Kubernetes 集群配置的切换过程。主要功能包括：
- 显示已配置的 Kubernetes 集群列表。
- 快速切换到指定的集群配置（支持数字选择或关键字模糊匹配）。
- 显示当前激活的集群名称、上下文，以及默认命名空间。
- 自动过滤损坏的配置文件，确保切换成功。

## 功能特性
- 支持主菜单实时刷新集群列表。
- 提供当前集群的精准哈希数据对比，防止重复切换。
- 提供简洁直观的用户交互体验。

## 安装教程
1. 将 `switch-kube.ps1` 文件下载到本地。
2. 将文件放置到任意路径，并确保 PowerShell 脚本运行权限已启用。

## 使用方法
1. 运行 `switch-kube.ps1` 文件以进入主工具菜单：
   ```powershell
   .\switch-kube.ps1
   ```
2. 按照菜单上的提示进行操作：
   - 输入数字选择集群。
   - 输入 `q` 退出工具。
   - 输入 `r` 刷新配置列表。

## 注意事项
- 环境配置文件需命名为 `config - 集群名` 格式。脚本会自动加载这些文件。
- 如果配置文件损坏，脚本会跳过并提示用户处理。

## 示例
运行后菜单展示：
```
==========================================================
           Kubernetes 集群快速切换工具
==========================================================
【指令说明】数字=选择集群 | q=退出 | r=刷新列表

[1] 集群A
[2] 集群B  (*当前集群)
==========================================================
当前生效集群：集群B
上下文名称：cluster-B-context
默认命名空间：default
==========================================================
请输入操作指令：
```
